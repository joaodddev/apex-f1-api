# 🏎️ Apex F1 API
API REST desenvolvida em Golang que consome dados da OpenF1 API e fornece endpoints organizados, performáticos e fáceis de usar para análise de dados da Fórmula 1.

# 🚀 Objetivo

Criar uma camada intermediária inteligente sobre a OpenF1 API, melhorando:
- organização dos dados
- performance
- legibilidade dos endpoints
- reutilização

# 🛠️ Tecnologias
- Go (Golang)
- Gin (HTTP framework)
- Resty (HTTP client)
- OpenF1 API
